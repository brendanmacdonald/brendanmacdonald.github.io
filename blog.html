How to take a text value of the format MM:SS and convert it to a Duration datatype.

16-03-2023

When I imported some data from the Parkrun UK website, the elapsed run time column with a value of 'MM:SS' (minutes and seconds) automatically had a datatype assigned of 'Text'. In order to use this column more accurately, I tried setting the datatype to 'Duration' in Power Query editor but it errored. To get around this error, I did the following:

  1. In Power Query editor, split the column on the semicolon. This resulted in two new columns I labelled 'Time Minutes' and 'Time Seconds'.
  2. Create a custom column with the following formula:
    • # #duration(0,0,[Time Minutes],[Time Seconds])
  3. Set the datatype of the custom column to Duration.
  4. Close and apply the changes.
  5. In the modelling view, set the custom column to have a datatype of Time and set the format to be h:nn:ss
